# ImageLens Pro - User Guide

## Welcome to ImageLens Pro

ImageLens Pro is an intelligent web-based image error detection and correction platform that analyzes uploaded images, identifies various types of errors, and provides interactive visual feedback with hover effects and color-coded highlighting.

## Getting Started

### 1. Sign In

- Visit the application homepage
- Click "Sign In" button
- Sign in with your Google account
- The first user to register automatically becomes an administrator

### 2. Dashboard

After signing in, you'll be directed to your dashboard where you can:

- View all your uploaded images
- See the status of each image (pending, processing, completed, failed)
- Access image analysis results
- Delete images you no longer need

## Uploading Images

### Step 1: Navigate to Upload Page

- Click "Upload" in the navigation bar
- Or click "Upload New Image" button on the dashboard

### Step 2: Select Your Image

You have two options:

**Option A: Drag and Drop**
- Drag an image file from your computer
- Drop it into the upload area

**Option B: Click to Browse**
- Click anywhere in the upload area
- Select an image from your file browser

### Supported Formats

- **File Types**: JPG, PNG, GIF
- **Maximum Size**: 5MB
- **Recommended**: High-resolution images for better error detection

### Step 3: Upload and Analyze

- Review the image preview
- Click "Upload & Analyze" button
- Wait for the upload to complete
- You'll be automatically redirected to the analysis page

## Analyzing Images

### Starting Analysis

1. After uploading, you'll see your image with a "pending" status
2. Click the "Analyze" button to start error detection
3. The system will process your image (this may take a few moments)
4. Status will change to "processing" then "completed"

### Understanding Error Detection

ImageLens Pro detects five types of errors:

#### 1. Spelling Errors (Red)
- Misspelled words
- Typos and common mistakes
- Example: "recieve" → "receive"

#### 2. Grammatical Errors (Orange)
- Subject-verb agreement
- Tense inconsistencies
- Example: "They was" → "They were"

#### 3. Spacing Issues (Yellow)
- Missing spaces
- Extra spaces
- Example: "alot" → "a lot"

#### 4. Context Errors (Blue)
- Wrong word usage
- Homophones
- Example: "their" → "there"

#### 5. Suggestions (Green)
- Improvement recommendations
- Better word choices
- Style enhancements

## Interactive Error Visualization

### Viewing Errors

- **Error Markers**: Colored dots appear on the image at error locations
- **Color Coding**: Each error type has a unique color for easy identification
- **Error Count**: See total errors detected in the header

### Hover for Details

**On the Image:**
- Hover over any colored dot
- A tooltip appears showing:
  - Error type
  - Original text
  - Suggested correction
  - Detailed description

**In the Side Panel:**
- Errors are grouped by type
- Click or hover over an error in the list
- The corresponding marker on the image will highlight
- Scroll through all detected errors

### Error Summary Panel

The right side panel provides:

- **Total Error Count**: Number of errors by type
- **Grouped Errors**: Organized by category
- **Quick Navigation**: Click any error to highlight it on the image
- **Detailed Information**: Full description and suggestions

## Managing Images

### Viewing Image Details

From the dashboard:
- Click "View" button on any image card
- See full analysis results
- Review all detected errors
- Access correction suggestions

### Deleting Images

From the dashboard:
- Click the trash icon on any image card
- Confirm deletion
- Image and all associated data will be permanently removed

## Tips for Best Results

### Image Quality

- **Use high-resolution images** for better text detection
- **Ensure good lighting** in the original image
- **Avoid blurry or distorted images**
- **Clear text** produces more accurate results

### File Size

- Keep images under 5MB
- Compress large images before uploading
- Use appropriate file formats (JPG for photos, PNG for text)

### Error Review

- **Review all suggestions** - not all may apply to your context
- **Hover over each marker** to understand the error
- **Check the description** for detailed explanations
- **Consider the context** of your content

## Troubleshooting

### Upload Issues

**Problem**: Upload fails
- Check file size (must be under 5MB)
- Verify file format (JPG, PNG, or GIF only)
- Ensure stable internet connection

**Problem**: Image not displaying
- Refresh the page
- Check browser console for errors
- Try uploading again

### Analysis Issues

**Problem**: Analysis stuck on "processing"
- Wait a few minutes (analysis can take time)
- Refresh the page to check status
- Contact administrator if issue persists

**Problem**: No errors detected
- Image may not contain text
- Text may be too small or unclear
- Image quality may be insufficient

### Display Issues

**Problem**: Error markers not showing
- Ensure analysis is completed
- Refresh the page
- Check that errors were actually detected

**Problem**: Tooltips not appearing
- Hover directly over the colored dots
- Ensure JavaScript is enabled
- Try a different browser
